# Crisis & Opportunity

A digital repository of early-20th-century sociology works by Black scholars, published at https://www.crisisopportunity.org. Static site, hand-curated, 61 articles (~261k words) plus PDF scans of originals.

## Repo layout

- `markdown/` — source content, one `.md` per article, with YAML frontmatter (author, title, citation). `index.md`, `about.md`, `art.md` are generated by `preprocess.ipynb`.
- `templates/` — pandoc templates. `article_html.template` for articles, `html.template` for index/about/art.
- `docs/` — built site served by GitHub Pages. **Do not hand-edit** — regenerated by pandoc. Contains `articles/`, `articles/PDFS/`, `books/`, `Images/`, `sitemap.xml`, `sitemap_articles.xml`, `robots.txt`, `CNAME`.
- `articles.csv`, `art.csv` — metadata driving the index build (title, author, journal, year, category, article_url, artpng).
- `preprocess.ipynb` — the build. Reads the CSVs, renders `markdown/index.md`, then runs pandoc to produce `docs/*.html` and `docs/articles/*.html`. Cache-busts CSS via `?id={random}`.
- `add_seo_metadata.py`, `add_lazy_loading.lua`, `optimize_images.sh` — post-processing helpers touched during the SEO/perf pass.
- `word/` — .docx exports (optional, commented out in the notebook).
- `build/` — currently empty.

## Build workflow

1. Edit source in `markdown/*.md` or update `articles.csv`.
2. Run `preprocess.ipynb` end-to-end. It regenerates `markdown/index.md` from the CSV and calls pandoc for every article + index/about/art.
3. Commit both the `markdown/` source and the regenerated `docs/` output. GitHub Pages serves from `docs/`.
4. There is no CI build step — `docs/` must be committed in its built form.

## Conventions

- Article URLs are stable slugs (e.g. `spawn.html`, `two_islands.html`). Don't rename — they're indexed by Google and linked externally.
- PDFs live under `docs/articles/PDFS/` with a `Author_Year_Topic.pdf` naming pattern.
- Categories (in `articles.csv`) follow a fixed display order set in the notebook: Racial Identity → White Racism → Great Migration → Labor → Gender → Health → Social Movements → Methods → Crime → Education → Family.
- All URLs should be `https://` — a pass in commit `d023623` cleaned up lingering `http://` references.
- Images are lazy-loaded (commit `b3ee61a`, via `add_lazy_loading.lua`).
- Sitemaps: `sitemap.xml` is a sitemap **index** pointing to per-section sitemaps (articles, books) — added in `eb98c00`.

## SEO context

Google Search Console started collecting impressions on **2026-04-12**. First 3-month snapshot (exported to `crisisopportunity.org-Performance-on-Search-2026-04-14/`) shows ~464 clicks / ~72k impressions / avg position ~8.5. Strongest pages: `spawn.html`, `pathology.html`, `frontage.html`. Biggest CTR-opportunity pages (high impressions, low CTR, position 6–10): `migration.html`, `social_organization.html`, `etiquette.html`, `two_islands.html`, `lilywhite.html`, `negro_problem.html`. `convict.html` is buried at position ~43 despite many convict-lease-system queries — content/title probably not aligned with intent.

## Things to avoid

- Don't hand-edit files in `docs/articles/` — rerun the notebook instead; hand edits will be overwritten.
- Don't rename article slugs or move PDFs — breaks external links and search rankings.
- Don't commit without rebuilding if you changed `markdown/` or `articles.csv`.
